Principais formatações

Negrito/Destaque

Frase com termo em negrito com a tag <b> não semântica

Frase com termo em negrito com a tag <strong> semântica

Itálico/Ênfase

Frase com termo em itálico com a tag <i> não semântica

Frase com termo em itálico com a tag <em> semântica

Texto marcado

Para criar um texto marcado utiliza-se a tag <mark>

Texto grande e pequeno

Com a tag <big> (tag obsoleta) temos um texto grande

Com a tag <small> temos um texto pequeno

Texto deletado

A tag <del> risca um texto

Texto inserido

Para sublinhar um texto utilizamos a tag <ins>

Existe tambem o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para fazer coisas como: x20 usamos a tag <sup>

Texto subscrito

Para fazer coisas como: H2O usamos a tag <sub>
